Command Line Freebase Search Sample Instructions

A starter application has been generated for you.

  1. Install Python 2.
  2. Download the starter application and unzip it. It will unzip into a directory called freebase-cmd-line-sample.
  3. Within the directory of files you just unzipped, run your starter application:

    python sample.py

Congratulations, you are up and running! At this point you will want to add calls into the Freebase Search to the sample.py file. Please read the sample.py file carefully, it contains detailed information in the comments. For more information on the Freebase Search Python library surface you can visit:

https://developers.google.com/resources/api-libraries/documentation/freebase/v1/python/latest/

The Freebase Search has been turned on in the Cloud Console. You can see and adjust the OAuth 2.0 project settings by visiting:

https://cloud.google.com/console#/project/484699115975/apiui

Also check out the Python Client Library documentation, and get more information on the Freebase Search at:

https://developers.google.com/freebase/
